## Encapsulation-of-thinking


>~~“ 本文不适合老司机… ”~~   





代码不会可以多敲几次，学习的重点是思想；思路都没有就去写代码这是没有意义的，也没有必须非得 第一步干嘛/第二步干嘛。  
多数情况下是采用逆向思维，先考虑你要干什么，缺什么补什么；要知道知识是学习不完的，时间是自己掌握的，而我们要做的是 [在学习知识中，提高自己--1.学习新知识的能力 2.(. . .)] 。
